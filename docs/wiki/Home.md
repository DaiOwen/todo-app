## 当前功能

### ✅ 已完成功能

| 功能 | 状态 | 说明 |
|------|------|------|
| 任务增删改查 | ✅ 完成 | 支持创建、读取、更新、删除任务 |
| 任务状态切换 | ✅ 完成 | 点击切换完成/未完成状态 |
| 按状态筛选 | ✅ 完成 | 全部/进行中/已完成三栏筛选 |
| 任务优先级 | ✅ 完成 | 高/中/低优先级，不同颜色标记，自动排序 |
| 截止日期 | ✅ 完成 | 设置截止日期，过期高亮提醒，今日到期橙色提醒 |
| 任务分类/标签 | ✅ 完成 | 自定义分类，支持按分类筛选，任务卡片显示分类徽章 |
| 关键词搜索 | ✅ 完成 | 按关键词搜索任务标题、描述、分类 |
| 批量操作 | ✅ 完成 | 批量标记全部已完成、批量删除所有已完成任务 |
| 任务描述 | ✅ 完成 | 支持添加任务详细描述 |
| 创建时间记录 | ✅ 完成 | 自动记录任务创建时间 |
| 响应式UI设计 | ✅ 完成 | 适配手机和桌面屏幕 |
| RESTful API | ✅ 完成 | 完整的JSON API接口 |
| H2数据库 | ✅ 完成 | 内置内存数据库，开箱即用 |

### 🖼️ 界面预览

项目采用紫色渐变背景，卡片式设计，简洁现代：

- 顶部标题区域
- 添加任务表单
- 筛选标签栏
- 任务列表卡片
- 空状态提示

## 技术架构

### 后端技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Java | 21 | 开发语言 |
| Spring Boot | 3.2.0 | Web框架 |
| Spring Data JPA | - | ORM框架 |
| H2 Database | - | 内存数据库 |
| Maven | 3.8.7 | 项目构建 |
| Lombok | - | 简化Java代码 |

### 前端技术栈

| 技术 | 用途 |
|------|------|
| HTML5 | 页面结构 |
| CSS3 | 样式布局 |
| Vanilla JavaScript | 交互逻辑 |
| Google Fonts | 字体 |

### 项目结构

```
src/main/
├── java/com/todolist/
│   ├── TodoApplication.java      # 应用入口
│   ├── controller/               # API控制器
│   ├── entity/                   # 数据实体
│   ├── repositories/            # 数据访问层
│   └── service/                  # 业务服务层
└── resources/
    ├── application.properties    # 配置文件
    └── static/                   # 前端静态文件
        ├── index.html
        ├── style.css
        └── app.js
```

## 贡献指南

欢迎提交 Issue 和 Pull Request！

### 开发环境要求

- JDK 21+
- Maven 3.8+
- IntelliJ IDEA (推荐)

### 本地开发

```bash
git clone https://github.com/DaiOwen/todo-app.git
cd todo-app
mvn spring-boot:run
```

访问 http://localhost:8080 即可使用。
